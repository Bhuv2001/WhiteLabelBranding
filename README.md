# White Label Branding System

## Tech Stack

- React (Vite)
- Tailwind CSS
- Supabase
- React Router
- React Hot Toast

## Features

- Upload Company Logo
- Select Login Theme
- Live Login Preview
- Save Branding
- Auto Load Branding
- Responsive Dashboard
- Supabase Storage Integration

## Setup

```bash
npm install
npm run dev
```

## Environment Variables

```
VITE_SUPABASE_URL=
VITE_SUPABASE_ANON_KEY=
```
